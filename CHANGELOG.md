# Changelog

All notable changes to Climent Ads Assistant are noted here. Updates ship roughly every two weeks.

## 0.1.0 — 2026-07-13

First public release.

- Runnable **demo console** (Providers, Results, Measurement, Benchmarks, Weekly changes) with synthetic data.
- **Read-only by design** — no provider writes, no Ads writes.
- Demo mode requires no credentials and touches no real accounts.
- Project docs: quickstart, demo & data-safety notes, security policy, contributing guide.
